# Automatic-LOD-Generator
 
 ## Introduction
 Automatic level of detail generator and mesh simplifier for Unity. This project is based on the [UnityMeshSimplifier](https://github.com/Whinarn/UnityMeshSimplifier/) repository. And it aims to make the creation of the LODs process easier (1-click) with basic UI elements.
